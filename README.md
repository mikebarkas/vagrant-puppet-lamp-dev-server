## Virtual LAMP Server for Development

Creates an Ubuntu LAMP server for development.

Uses VirtualBox to build with Vagrant and provisioned with Puppet and a little shell.

----

* Ubuntu 14.04 _Trusty_
* Apache2
* MySQL
* PHP5
* PHP Xdebug
* Composer
* Git
* Vim
* Tmux
* Zsh
* Drush
