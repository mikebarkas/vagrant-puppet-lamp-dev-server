## Virtual LAMP Server for Development

Creates a Drupal centric Ubuntu LAMP server for development.

Uses VirtualBox to build with Vagrant and provisioned with Puppet.

----

* Ubuntu 14.04 _Trusty_
* Apache2
* MySQL
* PHP5
* PHP Xdebug _(todo)_
* Webgrind  _(todo)_
* PHPUnit  _(todo)_
* Drush _(todo)_
* Composer
* Git
* Vim  _(todo)_
* Tmux
* Zsh
